# Portfolio (ES/EN) — JSON-driven

Portafolio personal construido en **HTML + CSS + JavaScript** con contenido **100% dinámico** desde `data.json` e **i18n (Español/Inglés)**.  
Diseñado para que puedas agregar/editar secciones, proyectos, certificaciones, CTFs, skills e imágenes sin tocar el código.

## ✨ Features
- ✅ **Contenido data-driven**: todo se edita desde `data.json`
- ✅ **i18n ES/EN** con toggle + persistencia en `localStorage`
- ✅ **Secciones dinámicas**: Interests, Skills, Experience, Activities, Projects, Certs, CTFs, Education
- ✅ **Proyectos con 1 o múltiples repos** (ej. Frontend/Backend)
- ✅ **Experiencia/Actividades con bullets** (puntos) configurables
- ✅ Certificaciones con **imagen + fecha (YYYY-MM)** + link clickeable
- ✅ Diseño responsive + navbar sticky + back-to-top
